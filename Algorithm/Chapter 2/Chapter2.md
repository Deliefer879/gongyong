# Chapter 2

비선형 문제 : 선형 자료 구조로 표현할 수 없는 대표적인 문제(계층적 문제, 순환 종속성 문제)  
</br>
트리 순회 방식에는 전위, 후위, 중위, 레벨 순서 순회가 있다.  
</br>
이진 검색 트리(BST)는 널리 사용되는 형태의 이진 트리로 다음과 같은 속성이 있다.  
* 오른쪽 자식 노드의 값 >= 부모 노드의 값 >= 왼쪽 자식 노드의 값  
</br>

트리 연산의 시간 복잡도는 O(log N)이지만 한쪽으로 극단적으로 쏠린 트리의 경우 O(N)이 될수도 있다.  
이러한 문제를 해결하기 위해서 균형 트리가 존재하는데 트리가 편향되지 않도록 높이를 조정하여 편향성을 줄인다.  
균형 트리에는 AVL트리, 레드-블랙 트리등이 있다.  
</br>
N-항 트리는 각 노드가 N개의 자식을 가질 수 있는 트리이다.  
</br>
힙은 우선순위 큐의 구현에 사용되는 배열로 구현한 완전 이진 트리이다.  
힙의 초기화를 위해 원소를 1개씩 삽입하는 경우 O(N log N)의 복잡도를 가지지만  
힙 생성 알고리즘 사용시 복잡도는 O(N)의 복잡도를 가짐.  
</br>
