# Data Struct

## 1. Tree
* PreOrder  : 전위 순회 (루트 -> 왼쪽 -> 오른쪽) <- 대부분의 공간 분할은 전위 순회로 생성
* InOrder   : 중위 순회 (왼쪽 -> 루트 -> 오른쪽)
* PostOrder : 후위 순회 (왼쪽 -> 오른쪽 -> 루트)
* LevelOrder: 레벨 순회
</br></br>

* Binary Search Tree: 왼쪽 자식 노드는 부모 노드보다 작고, 오른쪽 자식 노드는 부모 노드보다 큰 트리
* Quad Tree : 4개의 자식 노드를 가지는 트리(보통 지형 분할에 사용)
* Oct Tree : 8개의 자식 노드를 가지는 트리(보통 3d 공간 분할에 사용)

## 좌표계 변환 공식

```
 * 0 ~ 1 -> -1 ~ 1 : x * 2 - 1
 * -1 ~ 1 -> 0 ~ 1 : x * 0.5 + 0.5

ex) w = 800, h = 600

    Q. 0 ~ 800 -> -400 ~ 400 (x 축)
        1. 0 ~ 800 -> 0 ~ 1 : x / w
        2. 0 ~ 1 -> -1 ~ 1 : (x * 2) - 1
        3. -1 ~ 1 -> -400 ~ 400 : x * w / 2

    Q. 0~600 -> -300 ~ 300 (y 축)
        1. 0 ~ 600 -> 0 ~ 1 : y / h
        2. 0 ~ 1 -> -1 ~ 1 : (y * 2) - 1
        3. -1 ~ 1 -> -300 ~ 300 : y * h / 2
        4. -300 ~ 300 -> 300 ~ -300 : y * -1 (직교 좌표계는 위가 +)

간단한 방법
    x)
        0 ~ 800 -> 0 ~ 1 : x / w
        0 ~ 1 -> -400 ~ 400 : x - w/2

    y)
        0 ~ 600 -> 0 ~ 1 : y / h
        0 ~ 1 -> -300 ~ 300 : y - h/2
        -300 ~ 300 -> 300 ~ -300 : y * -1

```
