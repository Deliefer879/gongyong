## 4. 라이브러리
* 새 라이브러리 만들기  
새 프로젝트 (Window 데스크톱 마법사(라이브러리)) -> 정적 라이브러리(.lib) ->빈 프로젝트

* 라이브러리 자동화
  * 라이브러리 수정 자동화
    * 속성 -> 빌드 이벤트 -> 빌드 후 이벤트 -> 명령줄
  ```
  md   "..\\..\\CoreLib\\Include"   // 폴더가 없는 경우 새로 생성 (있으면 덮어쓰기)
  copy "*.h" "..\\..\\CoreLib\\Include"
  copy "*.inl" "..\\..\\CoreLib\\Include"
  copy "..\\..\\output\\CoreLib.lib" "..\\..\\CoreLib\\Lib"

  // fmod 라이브러리 복사
  ex) copy "..\\..\\Fmod\\lib\\fmod64_vc.lib" "..\\..\\CoreLib\\Lib"
  ```

  * 라이브러리 빌드 자동화
    * 빌드할 프로젝트(ex)Basic_Core) -> 참조 -> 참조 추가


 * 템플릿 내보내기(현재 프로젝트 세팅을 vs 템플릿으로 만들어 다른 프로젝트에 적용가능)
   * 프로젝트 -> 템플릿 내보내기 -> 내보낼 항목 선택 -> 내보낼 위치 선택 -> 완료
